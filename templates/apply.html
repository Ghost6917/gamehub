Applying for developer

At the moment, we don't have an official developer application.

If you would like to work on a feature or two, please head over to the repository.


Not sure what to do?
Our issues tab has plenty of planned features that you could work on.

If you are interested in joining GameHub as a developer, please join our discord and message zerointensity#0101.


Difference between contributor and developer?

Contributors are simply people who have made a commit to the repository. They don't need to meet any requirements or write out any application.


Developers help maintain the entire project. They must meet the developer requirements and are provided a developer GameHub account.

​

Developer Requirements

Anyone applying for developer should be proficent in at least one of the following.

  • - TypeScript
  • - Python and FastAPI
  • - HTML and TailwindCSS

Anyone applying for developer should also be relatively proficent in all of the following.

  • - Familar with industry standards (such as DRY principle)
  • - Command lines
  • - Npm or Pip
  • - Git and GitHub
